Quick start

If you are new in gav-gav.js start here

Installation

To install gav-gav.js you need to have Node js and git installed

If you have, follow the instructions below

git clone https://github.com/nulnow/gav-gav.js
npm install
node gav-gav.js serve

This will run your app on http://localhost:3000

You can also specify port by adding flag --port=PORT

Example:

node gav-gav.js serve --port=8080

In browser you will see the default gav-gav.js welcome page

Simple hello world app

To write hellow world app you will need:

  1. Write a controller
  2. Write a route

let's start with a first one

cd in your project folder created by commands above

cd gav-gav.js

Controllers in gav-gav.js can be easly generated with a built in command line tool

Just run in your project folder:

node gav-gav.js make:controller MyController

It will generate MyController.js file in your /Controllers folder

In MyController.js class write a method index that reutrns 'Hello world!'. File should look like this:

const Controller = require('./Controller');
class MyController extends Controller {
    index() {
        return 'Hello word!';
    }
}

module.exports = MyController;

Next you need to tell the app when to run whis method. Open /routes/web.js file and add line:

Route.get('/hello', 'MyController@index');

App will run index method of MyController when user open in browser <yoursite.com>/hello
