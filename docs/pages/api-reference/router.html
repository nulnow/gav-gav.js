Router class

Router is an abstraction to easly generate a list of routes

All it does - it takes this:

Route.get('/', 'TestController@index');
Route.get('/users', 'TestController@users');

And converts it into this:

[
    {
        methods: ['get'],
        path: '/',
        middlewares: [],
        action: 'TestController@index'
    },
    {
        methods: ['get'],
        path: '/users',
        middlewares: [],
        action: 'TestController@users'
    }
]

How it works:

The code below should give you the basic understanding of this magic

const router = new Route;
const fileContent = fs.readFileSync('./routes/web.js', 'utf8');
function generateRoutes(Router) {
    eval(fileContent);
    return Route.flush();
}
const routes = generateRoutes(router);
